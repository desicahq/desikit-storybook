# Welcome to Desikit
This is the Desica design system, located at [desicahq/desikit](https://github.com/desicahq/desikit) on Github. This repo is located at [desicahq/desikit-storybook](https://github.com/desicahq/desikit-storybook) on github.

### Add the registry
To get started, you'll need to be a member of the [desicahq](https://github.com/desicahq) on github. Authenticate with the [gpr](https://github.com/features/packages) by adding the following line in your `.npmrc` file:
```c
@desicahq:registry=https://npm.pkg.github.com
//npm.pkg.github.com/:_authToken=
```
Your `authToken` should be a personal access token with permission to install packages.

### Install desikit
To install Desikit, you'll need to run
```bash
# With npm
npm i @desicahq/desikit

# With yarn
yarn add @desicahq/desikit
```

### Import the components
Here's an example of how to import the components in a nextjs project
```js
import { Footer } from '@desicahq/desikit'

function myComponent () {
    return <Footer />
}
```
